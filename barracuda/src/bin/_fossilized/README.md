# Fossilized Binaries

Archived binaries that are no longer compiled by Cargo but preserved as
historical reference. These preceded the current validation and experiment
binary architecture.

**Not compiled:** These files are not listed in `Cargo.toml` `[[bin]]` entries.
They may reference APIs that have since changed.

**Why keep:** Fossil record of the evolution from standalone scripts to
the modular validation/experiment framework. Useful for understanding the
origin of current patterns.

**Date archived:** Various, 2026 Q1.
