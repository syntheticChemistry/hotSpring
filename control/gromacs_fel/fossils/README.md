# Fossilized Scripts

One-shot and development scripts used during ABG FEL system building.
Superseded by `nest-validate guidestone {finalize,validate,run}` pipeline.

Preserved for provenance — these show how systems were originally constructed.

| Script | Original Purpose |
|--------|-----------------|
| `.finalize_v1.7.sh` | v1.7.0 finalization (UPPER_WALLS + re-run) |
| `build_gh11_system.sh` | GH11 xylanase system construction |
| `build_subsite_systems*.sh` | Multi-subsite enzyme complex building (v1-v3 iterations) |
| `build_sugars.py` | CHARMM-GUI sugar topology generation |
| `generate_sugar_gros.py` | Sugar GRO coordinate generation |
| `generate_sugar_pdbs.py` | Sugar PDB coordinate generation |
| `run_epimer_metad.sh` | Epimer metadynamics launcher |
| `run_epimer_pipeline.sh` | Full epimer simulation pipeline |
